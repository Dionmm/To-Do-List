# To-Do-List
Simple To Do List

## Build
1. git clone https://github.com/kennethk91/To-Do-List.git
2. cd To-Do-List
3. npm install
4. (if you don't have nodemon) npm install -g nodemon
5. nodemon server/server or npm start
6. Server up at: http://localhost:3000/
